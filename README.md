## WHY — 우리는 왜 자꾸 돈으로 영어 실력을 사려고 할까?

해외여행을 다녀오는 비행기 안, 우리는 늘 굳게 다짐합니다. "올해는 진짜 영어 공부한다. 영어 너 죽었다." 하지만 12월이 되면 결국 죽어 있는 건 나의 결심뿐입니다.

한국이라는 환경에서 자연스럽게 영어를 사용하기란 불가능에 가깝습니다. 집에서 "우리 오늘부터 영어로만 말해보자!"라고 선언해 봐도 _"Hi, nice to meet you, How are you, Thank you, Bye"_ 로 끝나는 허무한 대화가 전부죠. 결국 우리는 다시 유튜브를 켜고 '2026 영어 공부법'을 검색합니다. 그러다 휘황찬란한 마케팅 문구에 홀려 어느새 영어 학습 앱 결제창에 지문을 찍고 있는 나를 발견합니다.

_'하루 10분이면 원어민이 된다고?'_  
여유롭고 유창하게 영어를 구사하는 내 모습을 상상하며 기꺼이 카드를 긁습니다. 시간과 노력을 들이는 대신, '돈'이 내 영어 실력을 해결해 주리라 믿으면서요.

---

## HOW — 물과 친해지듯, 가장 부담 없고 확실한 방법

수영을 배울 때 가장 먼저 하는 일은 숨을 참고 물과 친해지는 것입니다. 영어도 마찬가지여야 합니다. '공부'라는 압박을 버리고, 일단 재미있고 자연스럽게 스며들어야 하죠.

방법을 찾던 중, "이 방법으로 영어가 늘지 않으면 삭발하겠다"는 [한 유튜버의 영상](https://www.youtube.com/watch?v=Z6NXm05VC14)을 보았습니다. 그가 제시한 해답은 바로 **TED-Ed**였습니다.

5분 내외의 짧은 길이, 눈을 뗄 수 없는 훌륭한 영상미, 그리고 완벽한 스크립트까지. 이거라면 지루한 '공부'가 아니라 매일 즐기는 '콘텐츠'가 될 수 있겠다고 확신했습니다.

시청 → 학습지 → 핵심 구문 → 재시청 → 스크립트 복기 — 이 완벽한 5단계 사이클이 머릿속에 그려졌습니다.

---

## WHAT — 나를 위해 만들고, 당신에게 공유하는 자연스러운 흐름

문제는 '지속성'이었습니다. 매일 영상을 찾고, 스크립트를 발췌하고, 모르는 단어와 문법을 정리하는 수동 작업은 결국 우리를 지치게 만듭니다. 그래서 AI(Gemini)와 API를 연동해 이 모든 과정을 100% 자동화했습니다.

teding의 가장 중요한 목적은 **'내가 매일 공부하는 것'**입니다. 개발자인 제가 매일 아침 가장 흥미로운 영상을 하나 큐레이팅하고, AI가 완벽하게 정제한 학습지로 먼저 공부합니다. 그리고 이 훌륭한 자양분을 지인들과, 그리고 당신과 공유하기 위해 앱을 만들었습니다.

결국 언어는 시간과 노력을 투자해야만 자라납니다. 더 이상의 영어를 위한 무의미한 결제는 멈추세요. 지갑은 닫아두고, 매일 15분의 시간만 가볍게 투자해 보는 건 어떨까요?

## 5단계 학습 사이클

| 단계 | 이름                | 목적                                    |
| ---- | ------------------- | --------------------------------------- |
| 1    | 무자막 시청         | 자막 없이 소리에만 집중                 |
| 2    | 스크립트 확인       | 원문 독해, 단어/문장 수준 파악          |
| 3    | AI 학습지           | AI 생성 문제 풀기 (PDF 다운로드/업로드) |
| 4    | 핵심 표현·구문 분석 | 일상 활용 표현 5개 + 심층 문법          |
| 5    | 재시청              | 학습 후 다시 들으면 들린다              |

> 관리자(바로 나!)가 매일 TED-Ed 영상 1개를 등록하면, 모든 사용자가 동일한 영상으로 학습한다.  
> AI 학습 자료는 최초 1회 생성 후 전역 캐싱 — 두 번째 접속자부터 즉시 로딩.

## 주요 기능

- **오늘의 영상** — 매일 새 영상 1개. 선택 고민 없음.
- **AI 학습자료 자동 생성** — Gemini 2.5 Flash가 학습지·핵심표현·구문분석 자동 생성
- **보고또보고** — 과거 날짜별 영상 목록. 다시 공부하고 싶을 때.
- **스트릭 & 공유** — 연속 학습 일수 추적, 카카오톡/SNS 공유
- **학습 가이드** — 비로그인 공개 페이지, 공부법 설명

## Tech Stack

- **Framework**: Next.js (App Router), TypeScript
- **DB / Auth**: Supabase (PostgreSQL, Auth, Storage)
- **AI**: Google Gemini 2.5 Flash / Claude (Fallback)
- **UI**: Tailwind CSS, shadcn/ui
- **State**: URL SearchParams (SSoT — 새로고침해도 스텝 유지)
- **Deployment**: Vercel

## Getting Started

```bash
# 1. 의존성 설치
pnpm install

# 2. 환경변수 설정
cp .env.example .env.local
# .env.local 편집

# 3. 개발 서버
pnpm dev
# http://localhost:4000
```

### 환경변수

```env
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_PUBLISHABLE_KEY=
SUPABASE_SERVICE_ROLE_KEY=

GEMINI_API_KEY=
ANTHROPIC_API_KEY=       # Fallback

SUPADATA_API_KEY=        # 유튜브 스크립트 추출
ADMIN_SECRET=            # 관리자 API 인증
ADMIN_EMAIL=

NEXT_PUBLIC_KAKAO_JS_KEY=
NEXT_PUBLIC_APP_URL=https://eng.hololog.dev
```
