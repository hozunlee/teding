tedding — Brand & UI/UX Guidelines

Version: 1.0 (Urban Botanical Edition)

Concept: 차가운 도시, 내 책상 위에서 우아하게 자라나는 지식의 플랜테리어

1. Brand Identity (정체성)

**'tedding(테딩)'**은 본래 수확한 작물을 고루 펼쳐 말리는 농업 공정입니다. 우리는 이를 현대 지식인의 **'심플한 큐레이션'**으로 재해석합니다. TED-Ed 영상(원시 데이터)을 사람이 흡수하기 좋은 최적의 상태로 정제(tedding)하여 제공합니다. 유저는 그저 물을 주듯, 하루 15분의 루틴으로 내 안의 영어를 성장시키면 됩니다. 또는 ted를 하다.의 의미를 가집니다.

1.1 Slogans

Main Slogan: "밀도 높은 영어를 가장 심플하게 펼치다"

Sub Slogan: "TED-Ed의 지식을 AI가 심플하게 큐레이션합니다. 매일 하나씩, 내 안의 영어가 자라나는 감각을 경험하세요."

2. Visual Identity (시각 요소)

에듀테크 특유의 아동틱하거나 과도하게 팝(Pop)한 느낌을 배제합니다.
미니멀리즘과 '어반 보태니컬(Urban Botanical)'에 기반을 둡니다.

3. UI/UX Core Feature: 메인 대시보드 플랜테리어 (Planterior)

사용자가 앱을 켜자마자 마주하는 홈 화면(대시보드)은 차가운 UI 속에 살아 숨 쉬는 '디지털 화분' 역할을 합니다.

3.1 스트릭(Streak) 시각화: 'The Growing Ring & Pixel Plant'

모던하고 매끄러운 대시보드 UI 속에 클래식 픽셀 아트(Pixel Art) 감성의 식물을 배치하여, 외부 이미지 파일 없이 100% CSS/SVG 코드로 구현 가능한 레트로-모던(Retro-Modern) 믹스매치를 선보입니다.

UI 구조: 대시보드 중앙에 위치한 미니멀하고 세련된 원형 게이지(활동 링). 그 중심에 코드(CSS box-shadow 또는 SVG crispEdges)로 렌더링된 픽셀 식물이 자리합니다.

성장 로직 (Pixel Evolution):

Day 1 (8-bit 씨앗): 원형 게이지의 1/7이 부드럽게 채워지며, 중앙에는 아주 단순한 2x2 픽셀 형태의 귀여운 새싹(8-bit 감성) 픽셀 블록이 등장합니다.

Day 3 (16-bit 발아): 픽셀의 해상도가 높아지며 잎사귀가 2~3개 달린 16-bit 픽셀 아트 묘목으로 진화합니다.

Day 7 (1주 완성): 링이 한 바퀴를 완벽히 채우면(Growth Green 컬러 발광), 섬세한 도트(Dot) 그래픽으로 표현된 '몬스테라' 또는 **'올리브 나무'**가 픽셀 화분(Pot)에서 뻗어 나옵니다.

인터랙션: 매일 '오늘의 큐레이션(학습)'의 4단계를 모두 완료(Step 4)하고 홈으로 돌아왔을 때, 고전 게임의 레벨업(Level-up) 효과음 같은 가벼운 햅틱 진동과 함께 도트 그래픽이 '뿅!' 하고 한 단계 자라나는 CSS 픽셀 애니메이션(Step easing)이 실행됩니다.

'공부', '숙제' 같은 강압적인 에듀테크 용어를 지양하고, **'지식의 정제'**와 **'루틴한 성장'**에 초점을 맞춘 지적인 톤을 유지합니다.
